## 🛠️ System Architecture Diagram & Layout

This repository features a completely decoupled architecture, intentionally separating presentation view components, global styling token sets, and computational backend logic routers to isolate data matrices cleanly:

```text
nizams-choice-marketplace/
├── components/          # Reusable structural UI view elements
│   ├── FeedbackForm.jsx # Frictionless Connect Hub query ticket processor
│   └── Navbar.jsx       # Viewport navigation & light/dark token controller
├── styles/              # Global aesthetic rules and custom design layers
│   └── globals.css      # Custom glassmorphism variables & layout transitions
├── utils/               # Decoupled state routers & computational scripts
│   └── aiRouter.js      # Natural language string-matching AI router logic
├── .env.example         # Production environment template key references
├── index.html           # Live production entry hub workspace canvas
├── package.json         # Node runtime environment manifest definitions
└── README.md            # Complete hackathon engineering documentation
